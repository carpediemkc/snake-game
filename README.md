# 贪吃蛇游戏

一个使用 HTML5 Canvas 和 JavaScript 开发的经典贪吃蛇游戏。

## 游戏特点

- 多个难度等级
- 关卡解锁系统
- 分数记录
- 暂停/继续功能
- 响应式设计，支持移动设备

## 玩法说明

- 使用键盘方向键控制蛇的移动
- 吃到食物可以增加分数和蛇的长度
- 撞到墙壁或自己会导致游戏结束
- 通过达到目标分数解锁新关卡
- 每个关卡速度会逐渐加快

## 技术栈

- HTML5 Canvas
- JavaScript
- CSS3
